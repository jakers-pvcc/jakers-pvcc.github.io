Chapter 3, System Definitions

Term Definition
Operating System Software that controls the overall operation of a computer. Allows users to store and retrieve files, and offers an interface where a user can execute programs
Job Queue Internal registry that holds programs in preparation for execution
Interactive Processing Allows a program to carry a "dialogue" with user while in the process of execution
Time-Sharing An internal system that allows operating systems to service multiple users at a time
Multiprogramming A form of program execution in which time is divided into intervals and then the execution of a single program is restricted to one interval
Application Software Programs for performing tasks particular to the machine's utilization
System Software Performs tasks that are generally common to computers
Utility Software Programs that perform activities that are fundemental to computer installations. Not included in the operating system
Graphical User Interface (GUI) Represent items that can be manipulated (files, programs, etc.) in a visual form
Shell Older form of user interface that displays files or programs via text
Kernel Contains non-visual software components that perform very basic functions required by computers
Device Driver Software units that communicate with controllers
File Manager Software that coordinates the use of the system's storage facilities
Memory Manager Software in charge of coordinating the use of the system's main memory
Virtual Memory Memory created by the movement of "PAGES" that allows the computer to access information quickly
Boot Strapping Allows the computer to start by retrieving the operating system from mass storage each time the computer is turned on
ROM Section of main memory that is nonvolatile. Allows the CPU to access the operating system
Deadlock A problem that can occur between processes when two processes require resources that allocated to another
Spooling Operation to prevent deadlock that works by creating "fake access" to resources that don't exist. This way, the program can be tricked into thinking it has access, and can later be given proper access to those resources when they are available
Super-User An account with the highest level of priveledge on an operating system
Multi-Factor Authentication More secure than single-factor authentication. Requires two or more separate "keys" to access an account

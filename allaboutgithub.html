Q. What is a version control system?

A. A version control system is a piece of software that allows development teams to track and manage chages to their source code.

Q. What is Git?

A. Git is an open-source version control software that developers use to manage and track changes to their source code.

Q. What is Github?

A. Github is an online service that allows developers to collaborate on large projects.

Q. What is the difference between Git and Github?

A. While Git is software that only works between small groups of developers, Github is an online platform where documents and data can be shared across countries instead of office spaces.

Q. Who started Github and how was it made?

A. Github was started by Tom Preston-Werner, Scott Chacon, Chris Wanstrath, and P.J. Hyett. While Git was originally developed as an innovation for Linux, the creators of Github used Git as the base for their own development-sharing platform. This was done because Github's developers needed a much faster way to share and edit files.

Q. Who owns Github currently?

A. Github is currently owned by Microsoft.

Q. How much does a Github subscription cost?

Q.What is the Octocat?

The Octocat is an internal joke amongst Github developers. Part cat, part octopus, and 110% eating your code.
